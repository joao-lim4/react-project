### Installation

1. Clone o repo
   ```sh
   git clone https://github.com/joao-lim4/react-project
   ```
2. Entre no diretório
    ```sh
        cd react-project
    ```
3. Install NPM packages
   ```sh
    npm install 
   ```


## Usage

Inicie o projeto usando o comando abaixo

1.Start
```sh
    npm start
```
